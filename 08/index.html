Arrow functions

difference between normal functions and arrow functions

1. syntax const fun = () => {}
2. arguments keyword not available for arrow funciton, use spread operator
3. hoisting - moves variable, function, class, or import declarations to the top of their scope before the code is executed
hoisting does not work for arrow functions 4. This keyword
in normal funciton this keyword refers to the current object
in arrow function points to the window object as arrow func
this keyword always represents the object that defined the arrow function.
